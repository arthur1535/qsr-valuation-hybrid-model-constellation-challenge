# Models

This folder documents the model hierarchy used in the repository.

- `valuation/` contains the intrinsic value framework and shared valuation logic.
- `monte_carlo/` contains the probabilistic extension of the valuation.
- `lstm/` contains the standalone technical model.
- `hybrid/` contains the valuation-anchored hybrid workflow.

The key principle is that the technical models are subordinate to the valuation framework.
